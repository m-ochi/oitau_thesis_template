・コンパイラーですが「LuaLaTex」にして使ってください。これじゃないとエラーだらけになります。
・今までは、.clsファイルにパッケージを定義する際「\usepackege」を使っていたと思いますが、ここでは、「\RequirePackage」を使っています。意味はほとんど同じですがエラー回避のため.clsに定義する際は、RequirePackageに書き換えて定義してください。本文にパッケージを定義する場合は今まで通りusepackegeでもエラーが出ないと思います。
・コンパイルする際、たまにタイムアウトする場合があります。何回かコンパイルしているとタイムアウトすることなくコンパイルできるので気にしないでください。
・フォントパッケージなどを定義するとエラーが高確率で起きると思います。その際.clsファイルを修正してもいいですが、余白、1行あたりの文字数、1ページあたりの行数は大学指定のサイズになっているのでそこだけはいじらないようにお願いします。
